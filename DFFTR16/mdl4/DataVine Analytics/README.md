# DataVine Analytics - Machine Learning Summative Lab

## Project Overview

This project demonstrates the complete machine learning workflow using three different datasets to solve real-world business problems.

The notebook covers:

- Data preparation and preprocessing
- Dimensionality Reduction using PCA
- k-Nearest Neighbors (k-NN) Classification
- Recommendation System using Cosine Similarity
- Clustering using K-Means and Gaussian Mixture Models (GMM)
- Hyperparameter Tuning using GridSearchCV
- Model Evaluation and Visualization

---

## Business Problems

### 1. Wine Classification

Predict wine classes using chemical properties with a k-NN classifier optimized using PCA and GridSearchCV.

### 2. Agricultural Feed Recommendation

Recommend similar chicken feed types using PCA and cosine similarity.

### 3. Regional Crime Pattern Analysis

Identify crime patterns using K-Means and Gaussian Mixture Models after feature selection and PCA.
